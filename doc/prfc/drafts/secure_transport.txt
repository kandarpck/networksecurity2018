                    Playground Request For Comments

Abstract

   Create a PRFC (Playground RFC) to specify a protocol that provides
   confidentiality, message integrity, and mutual authentication over
   the Reliable Internetwork Playground Protocol (R.I.P.P.)

Status of this Memo

   This document specifies a Playgrounds standards track protocol for
   the Playground Experiment, and requests discussion and suggestions
   for improvements.  Distribution of this memo is unlimited.
